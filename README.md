# Student Social Media Usage & Mental Health Analysis
**Tool:** Power BI | DAX | Power Query

## Overview
Interactive dashboard analysing social media usage patterns, stress levels,
and mental health scores across students in multiple countries.

## Key Insights
- Students using 4+ hours/day of social media showed significantly higher stress scores
- Instagram and YouTube were the dominant platforms across all age groups
- Female students reported higher stress correlation with social media usage

## Features
- KPI cards, pie charts, stacked bar charts, donut charts
- Dynamic slicers for country, platform, gender, and age group
- DAX measures for stress index and engagement ratios

## Dataset
Public dataset — [https://www.kaggle.com/datasets/shivasingh4945/student-social-media-and-mental-health-impact]

## How to Open
Download `dashboard.pbix` and open with [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free).
